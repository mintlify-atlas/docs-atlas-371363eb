## What you'll install

Frappe is an open-source full-stack web framework. ERPNext is the world's leading open-source ERP, built on top of Frappe. This guide walks you through installing the complete stack:

<CardGroup cols={3}>
  <Card title="Frappe Framework" icon="layer-group">
    The Python/JavaScript web framework that powers ERPNext and all Frappe apps.
  </Card>
  <Card title="ERPNext" icon="building">
    Full-featured ERP covering accounting, inventory, HR, manufacturing, and more.
  </Card>
  <Card title="MariaDB" icon="database">
    The recommended database backend, configured for full Unicode (utf8mb4) support.
  </Card>
  <Card title="Redis" icon="bolt">
    In-memory store used for caching, queuing, and real-time updates.
  </Card>
  <Card title="Node.js & Yarn" icon="node-js">
    JavaScript runtime and package manager for building Frappe's frontend assets.
  </Card>
  <Card title="Nginx & Supervisor" icon="gear">
    Reverse proxy and process manager for production deployments.
  </Card>
</CardGroup>

## Quick overview of the installation process

<Steps>
  <Step title="Install system dependencies">
    Install Python, Node.js, Redis, MariaDB, Yarn, and wkhtmltopdf on your system.
  </Step>
  <Step title="Install frappe-bench">
    Install the `bench` CLI tool using pip, which manages your Frappe environments.
  </Step>
  <Step title="Initialize a bench">
    Run `bench init frappe-bench --frappe-branch version-15` to create your Frappe workspace.
  </Step>
  <Step title="Create a site">
    Run `bench new-site yoursite.com` to create a new Frappe site with its own database.
  </Step>
  <Step title="Install ERPNext">
    Run `bench get-app erpnext` and `bench --site yoursite.com install-app erpnext` to deploy ERPNext.
  </Step>
  <Step title="Go live (optional)">
    Run `sudo bench setup production` to configure Nginx and Supervisor for production use.
  </Step>
</Steps>
