Frappe is an open-source, full-stack web framework built on Python and JavaScript. It comes batteries-included with a database abstraction layer, REST API, authentication, role-based permissions, and a rich UI toolkit — making it the foundation for building complex business applications quickly.

ERPNext is a fully-featured, open-source ERP system built entirely on top of Frappe. It covers accounting, inventory, manufacturing, HR, CRM, project management, and more. Because ERPNext is just a Frappe app, installing it follows the same workflow as any other Frappe application.

## What this guide covers

This guide walks you through a complete, step-by-step installation of Frappe and ERPNext from scratch. It covers:

- **Ubuntu 18.04 and 20.04** — the primary supported Linux distributions
- **ArchLinux** — community-supported installation path
- **Production setup** — Nginx, Supervisor, SSL, and multi-site configuration

Each section uses real commands taken directly from the official installation scripts. No filler, no guesswork.

<Note>
  Use the correct Frappe and ERPNext version numbers for your deployment. This guide uses `version-15` as the example throughout. Replace it with your target version (e.g., `version-14`) wherever it appears.
</Note>
